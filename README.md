# Gerenciador de Tarefas – C#

Aplicação de console desenvolvida em C# para gerenciamento de tarefas via terminal, com foco em lógica de negócio, organização de código e operações CRUD.

## Funcionalidades
- Criar tarefas
- Listar tarefas cadastradas
- Editar tarefas existentes
- Remover tarefas
- Execução totalmente via terminal

## Objetivo do Projeto
Este projeto foi desenvolvido com o objetivo de praticar:
- Lógica de programação em C#
- Organização e clareza de código
- Separação de responsabilidades
- Estruturação de aplicações back-end simples

## Tecnologias Utilizadas
- C#
- .NET
- Git

## Como Executar
1. Clone este repositório
2. Abra o projeto em uma IDE compatível com C#
3. Execute a aplicação pelo terminal

Concluído – com possibilidade de evolução futura para uso de banco de dados ou API.
